### Create App
1. ```mkdir example``` (optional)
2. ```cd example``` (optional)
3. ```git init```
4. ```heroku apps:create example```

### Add changes

1. ```heroku login```
2. ```git add .```
3. ```git commit  -m "msg"```
4. ```git branch -M main```
5. ```git push heroku main```
6. ```heroku logs --tail```
